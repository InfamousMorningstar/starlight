# Starlight Tours - Educational Website

An interactive educational website documenting the Starlight Tours - incidents of systemic racism and police misconduct in Saskatoon during the 1990s and early 2000s.

## ğŸ“ About

This website was created as part of a group research project to raise awareness about the Starlight Tours, where Saskatoon police officers abandoned Indigenous men in freezing temperatures on the outskirts of the city, resulting in several deaths. The project aims to educate about systemic racism, police misconduct, and the ongoing need for justice and reconciliation.
